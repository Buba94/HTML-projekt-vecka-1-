Klassen Scanner

Startsida Tillämpning Hur man använder metoder

Tillämpning

Tillämpning av Klassen Scanner finns i paketet java.util.Scanner.

För att använda sig av Scanner måste man först importera paketet.

import java.util.Scanner;

För att använda sig av en Scanner ska man initiera ett objekt av klassen Scanner.

I exemplet nedan initieras Scanner-variablen läs.


Scanner läs;


Scannern behöver sedan kopplas till en text.

Detta kan göras på flera olika sätt.

För att läsa texten inmatad i kommandotolken:

läs = new Scanner(System.in);

För att läsa text från en sträng:

läs = new Scanner(”Alla kan lära sig att koda java!”);


Alternativt

String s = ”Du måste gå på Academy för att lära dig koda java!”; läs = new Scanner(s);

För att läsa från en fil som heter ”filnamn.txt” måste paketet java.io.File importeras.

För att läsa från filen skriver man:

läs = new Scanner(new File(”filnamn.txt”));
