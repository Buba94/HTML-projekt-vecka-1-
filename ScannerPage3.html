Hur man använder metoder

Säg att du har följande rad kod i ditt program:


Scanner läs = new Scanner(”Reg Bil: ABC 123”);


För att förtydliga: Scannern läs är kopplad till en sträng bestående av en rad av totalt 4 tokens. Man skulle kunna säga att scannern ”ställer sin markör” i början av strängen. För att kolla om det finns en token att läsa in till höger om markören kan man skriva:


boolean finnsDetEnToken = läs.hasNext();


Variabeln finnsDetEnToken får värdet true om det finns en token, vilket det ju gör, nämligen ordet Reg. För att hämta ut Reg som sträng kan man skriva:


Sträng ord1 = läs.next();


Variabeln ord1 får då värdet ”Reg”. För att läsa nästa ord skriver man:


Sträng ord2 = läs.next();


Variabeln ord2 får då värdet ”Bil:”. För att läsa ut bokstäverna i registreringsnumret:


Sträng regBokstäver = läs.next();


Variabeln regBokstäver får värdet ”ABC”. Nu är markören framför ”123”. Den kan läsas av som en sträng, men ännu lämpligare som ett heltal av typen int. För att kontrollera om nästa token är ett tal kan man skriva:


boolean ärDetEttTal = läs.hasNextInt();


Variabeln ärDetEttTal får i detta fall värdet true. För att läsa in talet till en variabeln av typen int kan man skriva:


int regNr = läs.nextInt();


För att kolla om det finns en till rad att läsa från, kan man skriva:


boolean finnsDetEnRadTill = läs.hasNextLine();


finnsDetEnRadTill får värdet false, då strängen enbart har en rad. Men vill man generera ett körningsfel kan man skriva…


String nästaRad = läs.nextLine();


.. för att försöka läsa ut nästkommande rad. Hade det funnits en rad hade den returnerats som en sträng till variabeln nästaRad.
